machina.traj package¶

  • This package is the bockbone of machina.
    • Environment and Algorithm are seperated by this package.
  • Traj class
    • epis sampled by sampler are changed into the class Traj.
    • Traj is the list of dict.
  • Iterator
    • Output is batch, a dict keys of which are MDP elements such as obs, acs, rews, etc.
    • Methods of iterate* are used for On-Policy algorithms.
    • Methods of random* are used for Off-Policy algorithms.
